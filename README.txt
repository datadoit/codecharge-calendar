Simple calendar program written in CodeCharge Studio version 4.3.0.7676 using PHP4 and MySQL4.

When publishing to your web server, the value in tbl_config/site_url must match your web server's URL.

Ex:  Publishing to WAMP at C:\wamp\www\CodeChargeCalendar, the site_url value should be CodeChargeCalendar
     Publishing to live site at http://www.codechargecalendar.com, the site_url value should be www.codechargecalendar.com

Default login:  admin / password
